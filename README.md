

Автоматизированная система генерации резюме на базе LaTeX с использованием Docker и GitHub Actions.

## Описание
Этот репозиторий позволяет генерировать профессиональное резюме в формате PDF. Система автоматически собирает документ при каждом обновлении файла `cv.tex` в директории `CV/`.

## Структура проекта
* `CV/` — содержит исходный файл резюме `cv.tex` и логотип `hse_logo.jpg`.
* `.github/workflows/` — конфигурация CI/CD (GitHub Actions) для автоматической сборки PDF.
* `Dockerfile` — среда сборки на базе Ubuntu, включающая все необходимые пакеты TeX Live.
* `choose_system.py` — утилита для выбора базового образа Docker на основе хеша имени.

## Как это работает
1. **Сборка:** При каждом `push` или `pull request` в ветки `main/master` срабатывает GitHub Action.
2. **Контейнеризация:** Docker собирает образ с необходимыми библиотеками LaTeX.
3. **Генерация:** Внутри контейнера выполняется `pdflatex`, который компилирует `cv.tex` в `cv.pdf`.
4. [cite_start]**Результат:** Готовый PDF-файл выгружается как артефакт сборки[cite: 1, 14].

## Локальная разработка
Для генерации резюме локально выполните следующие команды:
```bash
# Сборка образа
docker build -t resume_builder .

# Запуск сборки
docker run --name resume_container resume_builder

# Копирование готового PDF на хост
docker cp resume_container:/app/CV/cv.pdf .
